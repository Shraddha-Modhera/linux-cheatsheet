### Basic Linux Commands
```bash
ls -la  # List all files including hidden files with detailed info
pwd  # Print working directory
cd /path/to/dir  # Change directory
cd -  # Switch to the previous directory
touch file.txt  # Create an empty file
mkdir -p parent/child  # Create nested directories
rm -rf dir  # Remove a directory and its contents forcefully
cp -r dir1 dir2  # Copy a directory recursively
mv file1 dir/  # Move a file to a directory
find /home -name "*.log"  # Find files with .log extension
locate myfile  # Quickly find a file by name
```
